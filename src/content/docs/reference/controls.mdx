[Mapbox Controls Demo](https://korywka.github.io/mapbox-controls/preview/)

There a number of UI Controls that can be added via Mapbox APIs. The current ones we have are:

- Zoom Buttons.
- Search Input.
- Location Button.
- Directions Button for Driving / Walking / Cycling routes.
- Popups.

Additional controls available are: 

- Style changer button.
- Export button.
